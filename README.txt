Welcome to use this Music Recommendation System
ways to use this system:
1. read the provided song_Database.txt, which is our current songs database. Choose three songs out of this database.
2. open MySql and create the a database called music.create a table using sql
create table songs(id char(10),name char(200) not null, times int not null);
3. run main.java start the system. 
4. Input three song id into the java console based on the song_Database.txt in sequence.
5. get the output from the java console

Note: this system can not run another time unless a new table is added and the two sql in database.java to the new table name.
if your localhost MySql password is not null replace the password with your password.
